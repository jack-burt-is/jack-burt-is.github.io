Retail Store Game

My first major project in Unity was a retail/tycoon type game where the player is able to build up their own business from scratch. The trailer for which can be seen above. The game is far complete, and likely won't be revisited, due to a scope far too large for a solo project. Despite this, the project was a success, as not only did it teach me the basics of Unity, it also sparked a real interest in game development that lead me to start exploring further.

The project ended up not being particularly feature rich, but a rudimentary character creator was successfully implemented, as well as cut-scenes, dialog, object placement and movement, a night/day cycle through the use of a clock, and the beginnings of a monetary system. All art was created by myself. The workign features can be seen in the trailer on the following page.
